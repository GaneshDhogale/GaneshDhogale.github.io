# Ganesh Bhat - Portfolio Website

A modern, high-performance personal portfolio website built with semantic HTML, CSS3, and JavaScript. Designed with a premium "Matte Black + Electric Blue" aesthetic, focusing on Agentic AI and Automation expertise.

## Features
- **Modern Design**: Glassmorphism, dark mode, and smooth animations.
- **Responsive**: Fully optimized for mobile, tablet, and desktop.
- **High Performance**: Pure Vanilla JS/CSS (no heavyweight frameworks).
- **SEO Friendly**: Semantic structure and meta tags.
